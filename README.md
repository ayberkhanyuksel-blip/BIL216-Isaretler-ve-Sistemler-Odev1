# BIL216 İşaretler ve Sistemler 


 Görev 1: Sinüzoidal İşaretlerin Örneklenmesi
- Temel Frekans (f0): 103 Hz (38 + 63 + 02).
- Analiz: f1=103Hz, f2=51.5Hz ve f3=1030Hz frekanslarında sinyaller üretilmiş ve Nyquist kriterine uygun olarak örneklendirilmiştir.
- Görselleştirme: Sinyaller ve toplamları zaman domaininde grafiklenmiştir.

 Görev 2: DTMF Sinyal Sentezi ve GUI Tasarımı
Arayüz: Python Tkinter kütüphanesi kullanılarak interaktif bir telefon tuş takımı oluşturulmuştur.
Sentez: Her tuş basımında ilgili iki frekans (f_low + f_high) toplanarak ses üretilmektedir.
Analiz : Üretilen sinyallerin doğruluğu FFT (Fast Fourier Transform) kullanılarak spektrum üzerinde kanıtlanmıştır.
